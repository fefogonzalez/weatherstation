# weatherstation
Estacion metereologica utilizando Arduino e impresion 3d

Hardware utilizado
- NodeMCU v3 (ESP8266MOD)
- LCD 2.4" TFT SPI 240x320

Mapeo de puertos
NodeMCU     LCD
3v          VCC
3v          RESET
3v          LED
G           GND
D1          DC
D2          CS
D5          SCK
D6          SDO (MISO)
D7          SDI (MOSI)
